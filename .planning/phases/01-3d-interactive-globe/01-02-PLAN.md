react-globe.gl tooltip API:
- polygonLabel: accessor returning HTML string for native tooltip on hover
- onPolygonHover: callback(polygon, prevPolygon) for custom hover logic
- onPolygonClick: callback(polygon, event, coords) for click handling
- arcLabel: accessor returning HTML string for arc tooltip

react-globe.gl arc API:
- arcsData: array of { startLat, startLng, endLat, endLng, ... }
- arcColor: accessor for arc color
- arcDashLength, arcDashGap, arcDashAnimateTime: dash animation
- arcStroke: line thickness
- arcAltitude: height above surface (prevents z-fighting)
- arcLabel: tooltip on hover
